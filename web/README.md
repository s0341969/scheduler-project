# 專案說明

## 目前內容
- 本專案為可直接部署的單頁式品牌著陸頁。
- 採用 ASP.NET Core 最小靜態網站模式，主要頁面位於 `wwwroot/index.html`。
- 介面語系為繁體中文（zh-TW），設計重點為在地商家的信任建立與立即轉換。

## 技術決策
- 使用 `Microsoft.NET.Sdk.Web` 建立最小主機，透過 `UseDefaultFiles` + `UseStaticFiles` 提供靜態頁面。
- 不依賴外部 UI 框架，使用原生 HTML/CSS/JS，確保可維護性與載入速度。
- 視覺採手機優先（mobile-first）響應式版型，並含柔和 reveal/stagger 動畫。

## 頁面結構
1. Hero（價值主張、主要 CTA）
2. 品牌簡介
3. 核心服務卡片
4. 為什麼選擇我們
5. 地點與交通資訊（含導航按鈕與地圖預留區）
6. 服務承諾與常見問題
7. 最終 CTA
8. Footer（地址、電話、營業時間、版權）

## 可替換欄位
請優先替換以下占位內容：
- `[公司名稱]`
- `[公司核心價值主張]`
- `[一句話介紹服務與地點優勢]`
- `[公司類型]`
- `[服務一] [服務二] [服務三]`
- `[電話]`（並同步更新 `tel:` 連結）
- `[營業時間]`
- `LINE` 連結、`OG:url`、`OG:image`

## 建置
```powershell
dotnet build
```
