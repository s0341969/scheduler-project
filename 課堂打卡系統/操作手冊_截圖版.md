# 課堂打卡系統操作手冊（截圖版）

本手冊提供實際畫面截圖，協助教務人員與管理者快速上手。

## 1. 首頁總覽

首頁可查看目前課堂清單、打卡狀態、最近打卡紀錄，以及快速進入打卡頁或 QR Code 打卡板。

![首頁總覽](/G:/codex_pg/課堂打卡系統/docs/images/home.png)

重點操作：

- 點擊 `進入打卡頁` 讓學生直接填寫打卡資料
- 點擊 `QR Code` 開啟投影用打卡板
- 已登入管理者時，可直接在首頁切換 `開放打卡` / `關閉打卡`

---

## 2. 管理者登入頁

管理者先進入登入頁，再使用帳號密碼進入後台。

預設登入資訊：

- 帳號：`admin`
- 密碼：`ChangeMe123!`

![管理者登入頁](/G:/codex_pg/課堂打卡系統/docs/images/login.png)

建議：

- 正式環境請立即修改密碼
- 僅將後台帳號提供給教務或授權管理人員

---

## 3. 管理後台

登入後可進入後台，集中維護課程與課堂時段。

![管理後台](/G:/codex_pg/課堂打卡系統/docs/images/admin.png)

後台可執行的工作：

- 新增課程
- 編輯或刪除課程
- 新增課堂
- 編輯課堂時間、主題與開放狀態
- 匯出 Excel 出席紀錄
- 開啟 QR Code 打卡板

建議日常流程：

1. 新增或確認今日課堂
2. 開放該堂課打卡
3. 投影 QR Code
4. 課後匯出 Excel
5. 關閉打卡

---

## 4. 學生打卡頁

學生可從首頁或掃 QR Code 進入打卡頁。

![學生打卡頁](/G:/codex_pg/課堂打卡系統/docs/images/checkin.png)

學生需填寫：

- 學號
- 姓名
- 備註（可不填）

系統保護機制：

- 同一堂課同一學號只允許成功打卡一次
- 若課堂未開放，送出時會被拒絕

---

## 5. QR Code 打卡板

課堂現場可直接投影或列印 QR Code 打卡板，讓學生掃碼打卡。

![QR Code 打卡板](/G:/codex_pg/課堂打卡系統/docs/images/qr-board.png)

建議使用情境：

- 教室投影
- 印出貼在門口
- 現場快速導向打卡頁

---

## 6. Excel 匯出

在管理後台的課堂列表中，點擊 `Excel` 即可下載該堂課的出席紀錄。

匯出內容包含：

- 課程代碼
- 課程名稱
- 課堂主題
- 課堂時間
- 學號
- 姓名
- 打卡時間
- 備註

---

## 7. 課程與課堂維護建議

### 課程維護

- 先建立課程基本資料
- 課程代碼不可重複

### 課堂維護

- 每次上課前先建立課堂時段
- `結束時間` 必須晚於 `開始時間`
- 已有出席紀錄的課堂不可刪除

---

## 8. 常見使用順序

### 每日教務流程

1. 登入後台
2. 建立或確認今日課堂
3. 開啟課堂打卡
4. 顯示 QR Code 打卡板
5. 讓學生完成打卡
6. 課後匯出 Excel
7. 關閉打卡
