# Changelog

## 2026-04-24

- 建立 ASP.NET Core MVC 課堂打卡系統專案骨架。
- 新增課程、課堂時段、打卡紀錄資料模型與 JSON 檔案持久化服務。
- 新增課堂儀表板、打卡頁面、成功失敗提示與開放/關閉打卡功能。
- 重製版型與樣式，改為繁體中文教務風格介面。
- 補齊 `README.md`、`CHANGELOG.md`、`TODO.md`。
- 新增管理者登入與 Cookie 驗證，將管理功能移入後台。
- 新增課程與課堂時段後台，可新增、編輯、刪除與切換打卡狀態。
- 新增 Excel 匯出功能，支援下載單堂課出席紀錄。
- 新增 QR Code 打卡板，支援投影與快速進入打卡頁。
- 引入 `ClosedXML` 與 `QRCoder` 套件。
- 新增 `操作手冊.md`，整理一般使用者與管理者操作流程。
- 新增 `操作手冊_截圖版.md` 與實際系統頁面截圖。
- 新增學生帳號登入與本人打卡流程，打卡身分改由登入帳號決定。
- 新增動態 QR Code 驗證，QR 預設 5 分鐘失效並自動刷新。
- 新增班級 SSID 網段檢查、IP / User-Agent / 裝置指紋紀錄與可疑打卡標示。
- 管理後台新增可疑打卡檢視，Excel 匯出加入風險欄位。
- 修正學生登入與管理者登入表單的 `ReturnUrl` 驗證問題，避免空白隱藏欄位導致登入失敗。
- 新增課程班級欄位，首頁改為依 `一年級 101-110`、`二年級 201-210`、`三年級 301-310` 分組顯示課堂總覽。
- 舊版資料檔若缺少班級欄位，系統會在載入時自動補入可用班級代碼。
